# 🚗 Car Blog – Responsive Flutter Web App

A modern **Car Blog application built with Flutter**, focused on clean UI, smooth navigation, and **100% responsive design** across mobile, tablet, and desktop screens.

This project was built to practice real-world Flutter layout patterns, component reusability, and responsive web design.

---

## ✨ Features

* 📱 **Fully Responsive UI**
  Works seamlessly on **mobile, tablet, and desktop** using adaptive layouts.

* 🧭 **Smooth Section Navigation**
  Custom navigation bar with scroll-to-section behavior.

* 🧩 **Reusable Components**
  Modular widgets for navbar, footer, sections, and cards.

* 📰 **Blog & Trending Sections**
  Dynamic blog layout with featured and trending posts.

* 🎨 **Clean & Modern UI**
  Consistent theming, spacing, and typography.

* ⚡ **Optimized Flutter Web Layout**
  Handles wide screens and small devices gracefully.

---

## 🛠️ Tech Stack

* **Flutter**

* Flutter Web
* Responsive Layout Techniques (`MediaQuery`, adaptive widgets)
